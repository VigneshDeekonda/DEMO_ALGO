# Algorand Event Ticketing System

## Overview
The Algorand Event Ticketing System is a decentralized application designed to facilitate the buying and selling of event tickets in a secure, transparent, and efficient manner. By leveraging the Algorand blockchain, the system ensures that ticket transactions are immutable and verifiable, reducing the risk of fraud and scalping.

## Features
- **Secure Transactions**: All transactions are secured by the Algorand blockchain, ensuring the authenticity of ticket ownership.
- **Scalability**: Built to handle high volumes of transactions efficiently.
- **User-Friendly Interface**: An intuitive interface for both buyers and sellers to navigate the system effortlessly.
- **Real-Time Updates**: Users receive real-time notifications on ticket availability and transaction status.

## Getting Started
To get started with the Algorand Event Ticketing System, follow these steps:

### Prerequisites
- An Algorand wallet with ALGO tokens for transaction fees.
- Node.js and npm installed on your machine.
- Access to the Algorand network (TestNet or MainNet).

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/VigneshDeekonda/DEMO_ALGO.git
   cd DEMO_ALGO
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Configure the `.env` file with your Algorand wallet details.

### Usage
- Start the application:
   ```bash
   npm start
   ```
- Open your browser and navigate to `http://localhost:3000` to access the system.
